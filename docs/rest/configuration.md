# ⚙️ REST Configuration

| Key                    | Type   | Required | Description |
|------------------------|--------|----------|-------------|
| `qabase.rest.base-url` | string | ✅        | Base URL for the API (e.g. `https://gorest.co.in/public/v2`) |

**Example (`application.yaml`)**
```yaml
qabase:
  rest:
    base-url: "https://gorest.co.in/public/v2"
```
